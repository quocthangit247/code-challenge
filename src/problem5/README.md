## Prerequisites

- Node.js >= 20
- Docker

## Project setup

```bash
yarn install
```

```bash
docker compose up -d
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## API Documentation

Server will be running on [`http://localhost:3000`](http://localhost:3000) and you can access the API documentation on [`http://localhost:3000/api`](http://localhost:3000/api)
